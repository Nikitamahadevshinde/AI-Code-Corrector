# AI Code Corrector

AI Code Corrector is a full-stack web application that analyzes source code, detects errors, suggests corrections, and explains mistakes in simple language using Google Gemini AI.

## Features

* AI-powered code correction
* Simple error explanations
* React frontend
* Node.js and Express backend
* Google Gemini API integration
* User-friendly interface

## Tech Stack

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### AI Integration

* Google Gemini API

## How It Works

1. User enters code in the frontend.
2. Frontend sends the code to the backend.
3. Backend sends the code to Gemini AI.
4. Gemini analyzes and corrects the code.
5. Corrected code and explanation are displayed to the user.
